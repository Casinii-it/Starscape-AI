Item Classification

Item classification is how Starscape describes how easy items are to obtain. For example, a simple mine or ore would be Common, while a blueprint guarded by enemies could be Legendary. The game uses two types of systems to classify items: **Rarity** and **Tier**.

This wiki will, for the sake of clarity, call these two systems **Rarity** and **Tier**. While this page will not list every item and its rarity, this page will serve as a reference guide for users to easily identify rarity and how it is referred to on the wiki.

- The **Rarity** system colors the entire item icon's background, and is used to signify how rare an item is - with parameters ranging from cost to drop chance. **For wiki icons, this is represented by the background color.**

- The **Tier** system adds a triangle with Roman numerals to the icon, used to signify how advanced an item is statistically. **For wiki icons, this is represented by the overriden border color (if a tier exists).**

Note that by default, **Rarity** defines the border color of an item, unless otherwise overriden by a **Tier** designation, which chooses its own color. For example, a **Common** item that has a **Tier I** designation will use the preferred **Uncommon** border color. **This applies both in-game and on the wiki.**

While this page will not list every item and its rarity, this page will serve as a reference guide for users to easily identify rarity and how it is referred to on the wiki.

## Rarity

Toggle Rarities



| Icon | Name | Information |
| --- | --- | --- |
|  | Inferior | Inferior designations are used for items that provide little value, such as metal scrap or damaged items. |
|  | Common | Common designations are used for items that are not difficult to obtain, such as low-tier ore and **Basic** tier items. |
|  | Uncommon | Uncommon designations are used for items that are somewhat difficult to obtain, such as low-tier salvage and **Tier I** modules. |
|  | Rare | Rare designations are used for items that are difficult to obtain, such as mid-tier salvage and **Tier II** modules. |
|  | Very Rare | Rare designations are used for items that are very difficult to obtain, such as high-tier salvage and **Tier III** modules. |
|  | Legendary | Legendary designations are used for items that are extremely difficult to obtain, such as Vexnium ore or Drone Command Cores. |
|  | Ancient | Ancient designations are used for items that are only obtainable from Ancient structures and their respective loot. This rarity is part of the rarity group known as **Exotic** items and associated with **Tier X** modules. |
|  | Hybrid | Hybrid designations are used for items that are only obtainable or craftable from drone loot drops. This rarity is part of the rarity group known as **Exotic** items and associated with **Tier X** modules. |
|  | Dread | Dread designations are used for items that are only obtainable or craftable from pirate loot drops. This rarity is part of the rarity group known as **Exotic** items and associated with **Tier X** modules. |

## Tier

Toggle Tiers



| Icon | Name | Information |
| --- | --- | --- |
|  | Basic | Basic tier items are associated with the **Common** rarity when used to determine borders. This tier appears on only the Stratos. |
|  | I | Tier I items are associated with the **Uncommon** rarity when used to determine borders. This tier is used for items that are craftable without prerequisite versions of the item. |
|  | II | Tier II items are associated with the **Rare** rarity when used to determine borders. This tier is used for items that require a Tier I version to build off of. |
|  | III | Tier III items are associated with the **Very Rare** rarity when used to determine borders. This tier is used for items that require a Tier II version to build off of. |
|  | X | Tier X items are associated with the three **Exotic** rarities when used to determine borders **(Ancient, Hybrid and Dread)**. Whichever exotic rarity the tier is associated with is used to determine borders. This tier is used for exotic drops. |
