Ships

Ships make up the majority of gameplay in Starscape, with travel, combat, mining, and hauling centered around players using ships to accomplish tasks. This page serves as an unfiltered index of all ship-related information on the wiki.

* For all in-game controls, see the Controls page.
* For all the customization options, see the Editing Menu page.
* For more information on what each statistic means for the ships, see the Statistics page.

## List of all ships

Additional information can be found on each ship page. Information not included in this table: shield regen, deflection power, hull regen, armor rating, energy regen, warp speed, warp charge time, sensor strength, signal range, role bonuses, and design limitations.
