Controls

In Starscape, there are different control layouts for walking, flying ships, navigating menus, and moving about the game.

General Controls

|  |  |
| --- | --- |
| Universe Map | M |
| System Map | N |
| Inventory | I |
| Interact | F |
| Secondary Interact | C |
| Toggle GUI | Shift + Tab |

Character Controls

|  |  |
| --- | --- |
| Movement | WASD |
| Jump | Space |
| Sprint | Shift |
| Walk Slowly | Ctrl |
| Fire | L-Click |
| Zoom/Aim | R-Click (Hold) |

Large Ship Controls

|  |  |
| --- | --- |
| Increase Throttle | R |
| Decrease Throttle | F |
| Pitch Down | W |
| Pitch Up | S |
| Turn Left | A |
| Turn Right | D |
| Focus Camera | Shift |
| Roll Left | Q |
| Roll Right | E |
| Shift Up | T |
| Shift Down | G |
| Fire third type | 3 |
| Fire fourth type | 4 |

In-Ship Controls

|  |  |
| --- | --- |
| Full Stop | X |
| Quick Warp Menu | Space (hold) |
| Change View | V |
| Fire first type | L-Click (Hold Optional) |
| Fire second type | R-Click (Hold Optional) |
| Launch Item | 1 |
| Launch item | 2 |
